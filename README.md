# win_api

Library for Windows API calls.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     win_api:
       github: mjblack/win_api
   ```

2. Run `shards install`

## Usage

```crystal
require "win_api"
```
