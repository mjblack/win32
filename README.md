# win32

Library for Win32 API.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     win_api:
       github: mjblack/win32
   ```

2. Run `shards install`

## Usage

```crystal
require "win32"
```
